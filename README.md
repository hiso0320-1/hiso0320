# Cafe Serenity Landing Page

이 프로젝트는 카페 홍보를 위한 모던하고 세련된 랜딩페이지입니다.

## 실행 방법

1. `index.html` 파일을 더블 클릭하여 웹 브라우저에서 엽니다.
2. 모바일 화면에서도 잘 보이는지 확인하려면 브라우저 창 크기를 줄이거나 개발자 도구(F12)의 모바일 모드를 사용하세요.

## 파일 구조

- `index.html`: 웹사이트의 뼈대와 내용
- `style.css`: 디자인 및 애니메이션 스타일
- `script.js`: 모바일 메뉴 및 스크롤 애니메이션 로직

## 주요 기능

- **반응형 디자인**: 데스크탑, 태블릿, 모바일 모두 최적화
- **스크롤 애니메이션**: 스크롤 시 요소들이 부드럽게 나타나는 효과
- **모바일 메뉴**: 햄버거 버튼을 통한 모바일 네비게이션
- **섹션 구성**: Hero(메인), About(소개), Menu(메뉴), Gallery(갤러리), Location(위치)

## 다음 단계
